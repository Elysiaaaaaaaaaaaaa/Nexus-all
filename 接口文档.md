# 接口文档

## 认证
- POST `/api/v1/auth/register`
  - 请求：`{ username, email, password }`
  - 响应：`{ access_token, token_type, user }`
- POST `/api/v1/auth/login`
  - 请求：`{ username, password }`
  - 响应：`{ access_token, token_type, user }`

## 健康检查
- GET `/`
  - 响应：`{ message, version }`
- GET `/api/v1/health`
  - 响应：`{ status, service }`

## 工作流
- POST `/api/v1/work`
  - 认证：`Authorization: Bearer <token>`
  - 请求：`{ project_name, user_input, mode, video_duration?, modify_num? }`
  - 说明：根据项目 `workflow_type` 自动路由到 text2video 或 image2video
  - 响应：`{ success, message, end_session, project_name, session_id, session_data }`

## 项目
- POST `/api/v1/projects/new`
  - 认证：`Authorization: Bearer <token>`
  - 请求：`{ project_name, workflow_type }`
  - 说明：`user_id` 从 JWT 获取，无需传入
  - 响应：`{ success, project_name, workflow_type, session_id }`
- POST `/api/v1/projects/list`
  - 认证：`Authorization: Bearer <token>`
  - 响应：`{ success, projects }`
- POST `/api/v1/projects/history`
  - 认证：`Authorization: Bearer <token>`
  - 请求：`{ project_name }`
  - 说明：`user_id` 从 JWT 获取，无需传入
  - 响应：`{ success, chat_history, session_data }`

## 资源
- GET `/api/v1/test-video-placeholder`
  - 响应：占位视频文件
- POST `/api/user/avatar`
  - 请求：`multipart/form-data`，字段 `avatar`
  - 响应：`{ success, data: { avatar_url } }`
