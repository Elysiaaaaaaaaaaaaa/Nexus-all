# 项目摘要

- 前端（`/mnt/d/out/0131/2/Fronted-main`）：登录/注册、项目管理、交互式对话与视频预览。
- 后端（`/mnt/d/out/0131/2/Backend-test`）：认证、项目接口、文生视频工作流调度。
- 文生视频流程（text2video）：创意 -> 大纲 -> 分镜脚本 -> 视频生成；仅在分镜脚本阶段弹出修改确认，其余阶段通过“确认”推进。
- 项目接口鉴权：`user_id` 由 JWT 解析得到，前端无需传入。
- 图生视频流程（image2video）：对话 -> 分镜提示词 -> 生成首帧图片 -> 图生视频生成；无参考图时走文生图首帧。
